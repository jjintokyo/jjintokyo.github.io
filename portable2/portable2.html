October 2025: Revisiting an older OpenWrt build ...

This build wasn't getting much attention so I upgraded it with built-in speakers, an infrared remote to drive the radio easily (thanks to an Arduino nano with IR receiver soldered on it) and DAB!

YES DAB on a router, LOL, by connecting to a remote DAB server (the flat LEGO box in the picture), which could be located in a different room / floor, close to the antenna where reception is best. Then the client controls the DAB server and the audio is streamed back to the client. This is all done in the excellent welle-cli software.

Latest source in the portable2 folder.
