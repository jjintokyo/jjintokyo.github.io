JJ's ArduiPi Radio!
FM / DAB+ / INTERNET / MP3 Player

Powered by an Arduino Nano and a Raspberry Pi 3 Model B+


Full source code available here:   

https://github.com/jjintokyo
Make your very own ArduiPi Radio, it's fun and you'll be pleased with the result!
The radio has 40 internet radio presets and a random mode (random everything or by genre) that allows you to find and listen to new stations.

To build this radio you will need an Arduino Nano or equivalent (+ breadboard, leds, wires and a 4x20 LCD), a Raspberry Pi 3 Model B+ (regular Pi 3 is fine too), an RTL-SDR dongle for FM / DAB modes, a USB memory key to store your MP3 files, and some good sounding USB powered speakers...
A small self-powered USB hub will come in handy too and don't forget the remote control to drive the radio. Any cheap infrared remote should work, even the one from your TV!

You will have to update in radio.ino the codes to match your remote, before uploading the sketch to your Arduino board. The Arduino IDE has a built-in example to help you with that...

Last but no least, I made a web interface to fully control the radio from a smartphone, tablet or PC.
You will need to configure a web server on your Rasbperry Pi (I used Apache) to enable this feature.
Also the web interface allows you to easily update the 40 Internet Radio links and
watch over your Raspberry Pi's health :-)
